# ⚾ Moneyball

A data-driven job search engine that ranks opportunities based on fit and competitive edge.

## 🚀 Concept

Instead of blindly applying to jobs, Moneyball scores and ranks opportunities based on alignment with a candidate’s background.

## 🧠 Features

- Profile-based job matching
- Scoring algorithm (skills, tools, role alignment)
- Ranked job output
- Lightweight, fast execution

## ⚙️ Tech Stack

- Python
- Pandas
- BeautifulSoup (planned for scraping)

## 📊 Example Output

```
⚾ Running Moneyball...

Jobs found: 2

🔥 TOP MONEYBALL TARGETS:

1.20 | Customer Success Manager @ Test SaaS Co
→ SQL Tableau ARR churn customer success...

0.90 | BI Analyst @ DataCorp
→ Python SQL dashboards Power BI analytics...
```

Most job seekers apply blindly.

Moneyball introduces a structured, data-driven approach to opportunity selection.
